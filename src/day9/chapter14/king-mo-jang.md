# 예외 처리(Exception Handler)
1. 방법
   1. 직접 처리(try-catch로 감싸기)
   2. 예외전가(throws 던지기)
2. try-catch
   1. try():
      1. ()안에: 자동으로 리소스를 반납해야 하는 클래스를 기입 (ex. stream)
      2. {}안에: 예외 발생 가능성이 있는 코드들을 기입한다
   2. catch(): try{}에서 에외가 발생했을 시 처리하는 차선책을 적는곳
3. 빨간 줄이 에러인지 예외인지 구분하는 방법
정답 : 마우스를 가져가보고 if(exception글자 존재 -> 예외, else -> 에러)
4. 직접 예외처리하기 좋은 상황과 throws하기 좋은 상황 구분하는 방법
5. 정답 : 하나의 메서드 안에 처리해야 할 예외가 두 개 이상이면 -> throws, 아니면 직접 처리
